MANUAL: MÉTODOS DE STRING (Cadenas)

  • .upper() - Convierte toda la cadena a MAYÚSCULAS.
  • .lower() - Convierte toda la cadena a minúsculas.
  • .capitalize() - Convierte solo la primera letra a mayúscula.
  • .title() - Convierte la primera letra de cada palabra a mayúscula.
  • .strip() - Elimina espacios en blanco al principio y al final de la cadena.
  • .split(sep) - Divide la cadena en una lista usando un separador (por defecto espacios).
  • .join(lista) - Une una lista de cadenas usando esta cadena como puente o separador.
  • .replace(vieja, nueva) - Reemplaza una subcadena por otra nueva.
  • .find(sub) - Devuelve la posición donde empieza una subcadena (o -1 si no existe).
  • .count(sub) - Cuenta cuántas veces aparece una subcadena específica.
  • .startswith(prefijo) - Devuelve True si la cadena empieza con el texto indicado.
  • .endswith(sufijo) - Devuelve True si la cadena termina con el texto indicado.
  • .isalpha() - Devuelve True si todos los caracteres son letras (falla si hay espacios o números).
  • .isdigit() - Devuelve True si todos los caracteres son números.
